HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The paragraph element creates a new paragraph.
  • You can create ordered and unordered lists in HTML. You are currently reading an unordered list!
  • When inserting an image element into your webpage, ensure to include an alt attribute to improve SEO and assist the visually impaired users.
  • Use the anchor tag to link to create a hyperlink!

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border frames the element and can be seen by added size and color.
  • Select all of the elements in the HTML document with the * selector.
  • Set class level properties by using a . before the class name.
  • Select multiple HTML elements for a CSS property by separating them with a comma.
  • display: block assigns each element to start on a new line.
  • Use the box-shadow property to add a shadow to any element, like the sections in this page!
  • The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A command adds all file modifications in the current working branch to the staging area. The staging area informs Git that the changes will be added in the next commit.
  • git commit -m "commit message written as imperative commands in present tense."
  • git pull origin main command ensures our local branch is in sync with the base branch in GitHub. This let's us pull down any changes from the base branch.
  • git push origin "feature branch name" pushes our changes to the remote branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • In an 'if statement', the computer is looking to execture the logic of the 'truthy' condition.
  • Values that evaluate to "falsy" are: 0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false.
  • Use the console.log method to output a message to the web console by adding an argument.
  • Use Chrome DevTools to View console.log Output.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • In JavaScript and many other programming languages, arrays are zero-indexed and sequential.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • A function will only execute if it is called.
  • Math.random() returns a A floating-point, pseudo-random number between 0 (inclusive) and 1 (exclusive).
